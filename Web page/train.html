Training your network using transfer learning

Transfer Learning is the process of using a pretrained network and training it upon your dataset, for more information on why we use it please check out my blog post

Selecting which pretrained network to use

The main thing to keep in mind is that the resolutions of the pretrained model and of your images need to be the same so:

If your dataset is 256x256 you should use --resume lsundog256, if it is 512x512 or 1024x1024 you should use --resume ffhq512 / ffhq1024 respectively

If your dataset does not fit any of these resolutions then you will have to import a custom pretrained model, i recommend this guide.

Running the code

After selecting the appropriate pretrained model and setting --resume, you should run through the rest of the settings which are explained on colab and ensure that you understand them. The default settings should work for most cases.

Once everything is configured simply set the path variables appropriately and run the cell, this will probably take a long time. Ensure you follow the information on how to make sure the session stays connected

After the transfer learning has completed you will be left with your trained networks pkl files and some sample false images
